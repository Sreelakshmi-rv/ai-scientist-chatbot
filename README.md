# Kelly - AI Scientist Chatbot

A poetic AI scientist chatbot that responds to every question with skeptical, analytical poems about AI limitations and evidence-based suggestions.

## Features
- Responds exclusively in poetic form
- Questions broad AI claims and highlights limitations
- Provides evidence-based, practical suggestions
- Maintains professional, analytical tone
